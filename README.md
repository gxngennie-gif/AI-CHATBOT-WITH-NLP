#DESCRIPTION OF THE TASK:
*Project Title*:
Rule-Based AI Chatbot Using Python and NLTK

*Project Description*
This project was developed using Python in Visual Studio Code.
The objective of this project is to create a simple AI-based chatbot that can respond to user queries using Natural Language Processing techniques.
The chatbot uses a predefined knowledge base (dictionary) to respond to common questions related to:
Artificial Intelligence
Machine Learning
NLP
Python
Basic greetings
The chatbot processes user input, converts it into lowercase format, and matches it with stored responses.

*Tools & Technologies Used*:
#Programming Language
Python 3.13

#Code Editor
Visual Studio Code

*Libraries Used*:

1️.NLTK (Natural Language Toolkit)
Used for basic NLP processing
Downloaded tokenizer package (punkt)

2️.Random & String (Built-in Libraries)
Imported for possible future improvements

*How the Project Works*:

1️.User types a message.
2️.Input is preprocessed (converted to lowercase & cleaned).
3️. Program checks if input matches predefined knowledge.
4️. If match found → chatbot gives response.
5️. If user types “bye” → chatbot exits.
6️. If no match → chatbot says it does not understand.
